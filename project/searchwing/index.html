About the project:

We are building a low cost drone specially designed to support Search And Rescue (SAR) missions on the sea.

At the moment the main usage scenario is to support the rescue missions in the mediterran sea: private rescue organisations like Sea-Watch e.V., Sea-Eye e.V. or ResQship e.V. conduct civil search and rescue operations. We support these operations with our autonomous aerial rescue drone that can be launched and operated from a search vessel.

My contribution:

I built the first version of the object detection algorithm and pipeline that is used to detect vessels in distress and will be used in the first missions.
